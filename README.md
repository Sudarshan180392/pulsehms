# PulseHMS — Hospital Management System

A full-stack hospital management system built with React, TypeScript, 
and Express.js. Designed to streamline patient admissions, room/bed 
tracking, and ward-level operations.

🔗 Live Demo: https://pulsehms.vercel.app

---

## Features

- 🏥 Patient admission and records management
- 🛏️ Real-time room and bed availability tracking
- 🚨 Alarm/alert system for ward notifications
- 📋 Patient status dashboard
- 🔐 Role-based access (Admin / Staff)
- 📱 Fully responsive UI

---

## Tech Stack

**Frontend:** React 18, TypeScript, Tailwind CSS, Vite  
**Backend:** Express.js, Node.js  
**Deployment:** Vercel

Frontend runs on `http://localhost:5173`  
Backend runs on `http://localhost:3000`
